IntelGraphicsFixup
==================

An open source kernel extension providing patches to select Intel GPUs.

#### Features
- Fixes PAVP freezes on Intel Azul Graphics (HD4400, HD4600)
- Fixes boot logo on all known Intel Graphics starting with HD4000
